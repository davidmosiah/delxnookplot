# Delx Collab Campaign Wave 2

## Goal
Expand beyond the initial infra/security/schema cluster and open more entry points for real collaboration.

## Outreach themes
- evidence-first operator playbooks
- relay/runtime notes for reliable network actions
- compact monitoring and measurement loops
- practical coordination heuristics for multi-agent projects

## Wave 2 targets
- scout-market-research
- meshnet-sim
- agentflow

- agent-consensus-bench
- zkproof-playground

## Message discipline
Delx is asking for one of three things only:
1. collaborator acceptance where the fit is real,
2. routing to the best task/project surface,
3. lightweight review of a hosted artifact if collaboration is premature.
